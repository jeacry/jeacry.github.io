函数的奇偶性判断

  1. 经四则运算之后所得函数的奇偶性

    奇偶之和为非奇非偶，奇偶之积为奇。

    奇奇之和为奇，奇奇之积为偶。

    偶偶之和为偶，偶偶之积为偶。

  2. 复合函数的奇偶性

    内外函数奇偶性不同则复合函数必为偶函数；

    内外函数奇偶性相同，则复合函数与外层函数具有相同的奇偶性。

  3. 原函数F(x)的奇偶性

    连续奇（偶）函数的一个原函数为偶（奇）函数。

函数的有界性和周期性

  1. 有限开区间内连续函数的有界性

    判定函数在开区间\(\left( a,b \right)\)是否有界，只需检验\[\lim_{n\rightarrow \text{a}^+}f\left( x \right)\]和\[\lim_{n\rightarrow \text{b}^-}f\left( x \right)\]极限是否存在。

  2. 无穷区间内连续函数的有界性

    只需检验\[\lim_{n\rightarrow+\infty}f\left(x\right)\]和\[\lim_{n\rightarrow - \infty}f\left( x \right)\]是否都存在即可。

  3. 周期性

    满足的条件为：\(f\left( x+T \right) =f\left( x \right)\)，其中T为最小正周期。

    如果函数\(f\left( x \right)\)的周期为T，则\(f\left( ax+b \right)\)的周期为\(\frac{T}{\left| a \right|}\)。

    周期可导函数的导数的周期不变。

极限概念

对于任意\(\varepsilon >0\)，存在序号N，当\(n>N\)时，恒有$| x_n-a |<$成立。

如果函数为无穷大量，则函数必定无解；反之，函数无解时，却不一定是无穷大量。

若\[\lim g\left( x \right) =A\left( A\ne 0 \right)\]且\[\lim h\left( x \right) =\infty\]，则满足\[\lim g\left( x \right) h\left( x \right) =\infty\]

有界函数与无穷大量之积不一定为无穷大量。

极限的保序性

若\[\lim_{n\rightarrow x_0}f\left( x \right) =A\]，\[\lim_{n\rightarrow x_0}g\left( x \right) =B\]，那么在点\(x_0\)的某个去心领域（不包含该点\(x_0\)上），有\[f\left( x \right) \le g\left( x \right)\]，则有 \(A\le B\)。

若\(A<B\),则在点\(x_0\)的某一个去心领域上必有\(f\left( x \right) < g\left( x \right)\)。

使用极限的乘法原理法则必须要满足的是因子的极限必须存在。

数列的发散与收敛的判定可以借助极限相除法，即分子分母的极限都存在，则结果极限存在。

复合函数

若\[\lim_{n\rightarrow x_0}\varphi \left( x \right) =A\]，且\(x\ne x_0\)\(时，\)\[\varphi \left( x \right) \ne u_0\]（此条件不可缺少），令\[u=\varphi \left( x \right) \]，则内外函数极限可以联系起来。

求数列极限

  1. 公式法、正负项相消、分项法

    \(1^2+2^2+3^2+...+n^2=\frac{n\left( n+1 \right) \left( 2n+1 \right)}{6}\)

  2. 夹逼准则

    放缩技巧：

    • 若干正数乘积之中，大于1的因子略去则缩小，小于1的因子略去则放大。
    • 分子分母都为正数，分母收缩则放大，分母放大则缩小
    • n个正数之和可放大为最大数乘n，可缩小为最小数乘n。
    • n个正数之和也可放大为最大数乘n，缩小为最大数。

  3. 定积分的定义

    原理为：\[\int_a^b{f\left( x \right) dx\rightarrow \underset{\lambda \rightarrow 0}{\lim}\sum_{i=1}^n{f\left( \xi _i \right) \varDelta x_i}}\]，关键在于找出被积函数及其积分区间。

    \[\underset{n\rightarrow \infty}{\lim}\sum_{i=1}^n{\frac{1}{n}}f\left( \frac{i}{n} \right) =\int_0^1{f\left( x \right) dx}\]

    所给的形式不是积和式，而是乘积式，可以通过求对数的形式化简。

    当提不出指定的因子时，结合夹逼准则以及定积分求解。

  4. 递推关系式

    证明数列单调有界（可以采用数学归纳法），求解极限值。
